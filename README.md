## Catani - aprendendo c++
Programas que escrevi aprendendo estruturas de dados em c++.
Separado em diretórios de aulas das matérias de Estruturas de Dados na UPF.

Os melhores tem binários na pasta ./bin.

Formatação de código que sigo:

Em grandes blocos de código entre chaves, as chaves devem sempre ter sua própria linha.  
O "else" sempre em nova linha.  

ex.:  

```
int main ()
{
    ...
    codigo
    ...

    return 0;
}
```
  
Adiciono espaços sempre que possível:

```
if (cond) {
    ...
    ...
}
else if ( condicao_muito_grande_nos_parenteses )
    codigo_de_uma_linha;
else { 
    ...
}

for ( int i = 0; i < n; i++ )
{
    ...
    ...
}
```
Como visto:
Quando o texto dentro dos parênteses é muito longo, leva espaços antes e depois.  
Em IFs, blocos de uma só linha não levam chaves.
